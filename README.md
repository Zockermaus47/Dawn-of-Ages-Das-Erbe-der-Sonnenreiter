# Dawn of Ages: Das Erbe der Sonnenreiter

## Project Overview
Dawn of Ages is an exhilarating survival game developed using Unreal Engine 5.7. Set in a vast, immersive world, players must navigate the challenges of survival while exploring rich landscapes, crafting tools, and engaging with both enemies and friends.

## Features
- **Open World Exploration**: Discover various biomes, each with unique resources and challenges.
- **Crafting System**: Create weapons, tools, and structures to aid survival.
- **Dynamic Weather & Day/Night Cycle**: Experience realistic environmental changes that affect gameplay.
- **Multiplayer Support**: Team up with friends or compete against others in a shared world.

## Installation Instructions
1. **Download the Game**: Clone the repository or download the ZIP file.
2. **Install UE 5.7**: Ensure you have Unreal Engine 5.7 installed on your machine.
3. **Open the Project**: Launch Unreal Engine and open the project file (`.uproject`).
4. **Compile the Game**: Build the project from the Unreal Engine editor.
5. **Run the Game**: Start the game from the editor or package it for distribution.

## Multiplayer Setup
To set up multiplayer in Dawn of Ages:
1. **Host a Game**: Choose the "Host" option from the main menu.
2. **Invite Friends**: Use the in-game invite system or share your IP address with friends to join your game.
3. **Configure Server Settings**: Adjust settings for gameplay experience such as maximum players, game mode, and difficulty.
